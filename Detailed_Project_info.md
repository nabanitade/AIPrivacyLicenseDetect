## 🎯 What Is This?

This library enables **all AI companies** (OpenAI, Meta, Anthropic, Perplexity, Google, X, and more) to automatically detect and respect **AI Privacy Licenses** when crawling websites, processing user queries, or building training datasets. It works in real-time to identify content restrictions and communicate them clearly to users across **all AI labs' training, ingestion, and crawling pipelines**.

### 📋 What This Library Does / Does Not Do

| This library **does**                                                                                     | This library **does not**                    |
| --------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| Detect machine-readable AI Privacy License links via robots, headers, meta, JSON-LD, or common file paths | Parse Terms of Service or free-text policies |
| Parse a few structured fields: Allow-Training, Attribution-Required, Allow-Commercial                     | Provide legal advice or risk scoring         |
| Return a small JSON-friendly result                                                                       | Crawl site content or ignore robots          |
| **Comprehensive coverage**: Websites, code repos, GitHub, Hugging Face, Kaggle, datasets, and more        | Make legal decisions or provide risk assessment |

**Scope**: This library detects *machine-readable* AI Privacy License declarations via HTTP headers, `<meta>` tags, JSON-LD, `robots.txt`, and common license file paths. It **does not** parse Terms of Service or free-text policies, score legal risk, or decide whether to crawl content. It returns a compact, JSON-friendly result you can plug into your own policies.

**Security**: For SSRF protection, this library only follows HTTP/HTTPS URLs on standard ports (80/443). Non-standard ports (8080, 3000, etc.) and private IP ranges are automatically blocked.

## 🔍 Detection Capabilities

### **5 Detection Methods (All Included)**
- **🤖 robots.txt** - `ai-privacy-license-link:` declarations with comment support
- **📡 HTTP Headers** - `ai-privacy-license-link` and `Link: rel="license"` headers
- **🏷️ HTML Meta Tags** - `<meta name="ai-privacy-license-link">` and `<link rel="license">`
- **📊 JSON-LD** - Structured data license declarations (limited scan for performance)
- **📄 License Files** - Common paths including `/.well-known/ai-privacy-license`
- **License Files** - `/license/AiPrivacyLicense-1.0`, `/LICENSE.txt`

### **Advanced Security Features**
- **🛡️ SSRF Protection** - Blocks private IPs, odd ports, credentials in URLs
- **🔒 Redirect Validation** - Validates every redirect hop for security
- **⚡ DNS Resolution Check** - Optional DNS lookup for enhanced SSRF protection
- **🚫 Content-Type Guards** - Only processes text/html/json content
- **📏 Size Limits** - Configurable max bytes per response (default 128KB)

### **Smart Pattern Recognition**
The system understands restriction language with enhanced parsing:

```
❌ "Training AI models is prohibited" → allow_training: false
❌ "Do Not Train Flag: strict" → allow_training: false
❌ "We opt out of AI training" → allow_training: false
📝 "Attribution Required: yes" → attribution_required: true
🎓 "Commercial Use Permitted: no" → allow_commercial: false
```

### **Production-Ready Features**

## 🔧 Configuration

### **Detector Configuration**
```python
from ai_privacy_license_detector import AIPrivacyLicenseDetector

# Full configuration options
detector = AIPrivacyLicenseDetector(
    timeout=15,                    # Request timeout in seconds (default: 10)
    user_agent="YourCompany-AI/1.0",  # Custom User-Agent
    verbose=False,                 # Enable detailed logging
    insecure_ssl=False,           # Disable SSL verification (dev only)
    max_bytes=128*1024,           # Max response size (default: 128KB)
    trust_env=False,              # Respect proxy environment variables
    max_redirects=5,              # Maximum redirects to follow
    enable_dns_check=True         # Enable DNS resolution for SSRF protection
)

# Context manager (recommended for resource cleanup)
with AIPrivacyLicenseDetector() as detector:
    result = detector.check_website("https://example.com")
```

```bash
# All available CLI options

  --timeout 30 \                    # Request timeout (default: 10)
  --max-redirects 10 \              # Max redirects (default: 5)
  --max-bytes 256000 \              # Max response size (default: 128KB)
  --user-agent "Custom-Agent/1.0" \
  --concurrency 8 \                 # Parallel requests (default: 1, max: 50)
  --respect-proxy \                 # Use HTTP_PROXY/HTTPS_PROXY
  --insecure-ssl \                  # Disable SSL verification (dev only)
  --verbose \                       # Enable detailed logging
  --ndjson \                        # Newline-delimited JSON output
  --compact \                       # Compact JSON (no indentation)
  --summary \                       # Print summary to stderr
```
