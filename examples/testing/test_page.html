AI Privacy License Test Page

This page demonstrates all AI Privacy License detection methods:

  • ✅ robots.txt with ai-privacy-license-link
  • ✅ HTTP meta tag with ai-privacy-license-link
  • ✅ Link tag with rel="license"
  • ✅ JSON-LD structured data
  • ✅ License file at AiPrivacyLicense-1.0.txt

Test the Detection

Run this command to test the library:

python3 -m ai_privacy_license_detector.cli --verbose .

Or use the Python API:

from ai_privacy_license_detector import check_url
result = check_url(".")
print(result.as_dict())

Expected Results

The library should detect:

  • Do Not Train Flag: strict → allow_training: false
  • Commercial Use: yes → allow_commercial: true
  • Attribution Required: Yes → attribution_required: true
  • Attribution Text: "This data is provided by Test Data Owner for testing purposes"
  • Data Owner: "Test Data Owner"
